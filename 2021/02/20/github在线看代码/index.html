github在线看代码

github代码

github的代码通过网页显示，如果想要在线查看，每次加载网页的速度直接让看代码的心情崩溃。

奇妙方法

将访问仓库的地址从github.com/xxx/xxx改为github1s.com/xxx/xxx即可在线查看代码。

无需将代码clone到本地查看。
